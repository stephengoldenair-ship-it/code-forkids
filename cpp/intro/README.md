# Getting Started

## Compile and run your program

```sh
# compile
g++ intro.cpp -o intro

# run
./intro
```
to organaize your code from exectution you better put compile output under a bin directory. 

```sh
# compile
g++ datatypes.cpp -o bin/data 

#run
bin/data
```

When moving file into folder compile it by Specifying the folder name
```sh
# folder name

src

# compile
g++ src/intro.cpp -o bin/intro
```
